Система ведение учета книг в библиотеке.
Есть 2 типа пользователя user и admin, которые отличаются в зависимости от паролей для входа.
admin - может добавлять книги в библиотеку и удалять их если они не находятся у пользователя.
user - может брать книгу и смотреть список всех книг в наличии в библиотеке(за исключением тех которые уже находятся у него)

Так же необходимо реализовать возможность переходить из одного меню в другое.
Т.е из меню user можно перейти в меню просмотра всех книг и из меню просмотра всех книг необходимо сделать возможность вернуться в предыдущее меню,
т.е основное меню user и.д.
